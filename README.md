GrammarDice
========================

Features
------------------------

#### Main features:

The Dice
------------------------

Each die is made up of 6 faces except for the consonants die which has 20 faces.
Some of the letters are repeated and some are not present. The following list is a compilation of the dice
faces.

#### Vowels

*U* = A
*D* = I
*L* = E
*R* = E
*F* = O
*B* = U

#### Numbers

*U* = 2
*D* = 2
*L* = 3
*R* = 3
*F* = 4
*B* = 5

#### Parts of speech

*U* = Adverb
*D* = Prepositions
*L* = Pronoun
*R* = Adjective
*F* = Interjection
*B* = Conjunction

#### Punctuation marks

*U* = ; (semicolon)
*D* = :    (colon)
*L* = ?    (question mark)
*R* = ,    (comma)
*B* = ()(parenthesis)
*F* = ""(quotation mark)

#### Consonants

The consonants which are present in the normal mode are:
_BCDFGHJKLMNPRSTVW_

Optionally You can challenge yourself by adding these four letters into the mix.
_QXYZ_
